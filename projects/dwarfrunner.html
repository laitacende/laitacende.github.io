Student project: Dwarf Runner

I was a part of a team which developed a field game (which we later called Dwarf Runner) for web and mobile. The server side was written in Java, while mobile version was developed for Android. The project was a part of a course at my university.

As a part of Frontend & Design team I took part in designing the whole look of the game, both on mobile and web. We created a prototype in Figma. Particularly, I designed the logo and the look of markers on a map.

The website is written using HTML5, CSS3, Sass and Javascript with library Websock to communicate with backend. There, I mainly implemented updating lobby list and the game itself.
